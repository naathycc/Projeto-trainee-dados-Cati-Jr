# Projeto Trainee em Dados - CATI Jr.

Este repositório contém o desenvolvimento do meu Projeto Final como Trainee de Dados da CATI Jr.

Para este projeto, foi utilizado o dataset **House Prices: Advanced Regression Techniques**, disponibilizado pela plataforma Kaggle. O conjunto de dados reúne informações sobre características de imóveis residenciais, contendo diversos atributos que podem influenciar o valor de venda de uma propriedade.

O objetivo do projeto é desenvolver e avaliar modelos de Machine Learning capazes de prever o preço de imóveis com base em suas características, aplicando na prática todo o fluxo de um projeto de Ciência de Dados.

Durante o desenvolvimento do projeto serão realizadas etapas como análise exploratória dos dados, tratamento e limpeza dos dados, treinamento de modelos preditivos e avaliação de desempenho.

As principais tecnologias utilizadas serão:

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn
* Streamlit
