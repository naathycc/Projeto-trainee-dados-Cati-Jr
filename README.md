# 🏠 Projeto Trainee em Dados - CATI Jr.

Este repositório contém o desenvolvimento do meu **Projeto Final** do Processo Trainee de Dados da **CATI Jr.**

## Sobre o projeto

Para este projeto foi utilizado o dataset **House Prices: Advanced Regression Techniques**, disponibilizado pela plataforma Kaggle. O conjunto de dados reúne informações sobre características de imóveis residenciais, contendo diversas variáveis que influenciam o preço de venda dos imóveis.

O objetivo do projeto foi desenvolver um fluxo completo de Ciência de Dados, desde a análise e preparação dos dados até o treinamento e avaliação de modelos de **Machine Learning** capazes de prever o preço de venda de imóveis.

---

## Etapas realizadas

- Análise Exploratória dos Dados (EDA)
- Limpeza de dados
- Engenharia de Features
- Seleção das variáveis mais relevantes
- Treinamento e comparação entre modelos supervisionados
- Avaliação utilizando Hold-out (treino e teste)
- Aprendizado Não Supervisionado (K-Means)
- Desenvolvimento de um simulador de preços com Streamlit

---

## Modelos utilizados

- Regressão Linear
- Decision Tree Regressor
- Random Forest Regressor

Após os experimentos, o **Random Forest** apresentou o melhor desempenho e foi escolhido como modelo final para o simulador.
